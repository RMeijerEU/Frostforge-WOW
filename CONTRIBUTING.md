# Contributing to Frostforge

Frostforge is a guided World of Warcraft dungeon campaign project.

## How to contribute

1. Pick a task from GitHub Issues or Discord.
2. Create a branch or fork.
3. Make your changes.
4. Open a Pull Request.
5. Wait for review before anything is merged.

## What you can contribute

- Campaign writing
- Quest text
- Lua scripts
- SQL changes
- Website articles
- Changelogs
- Testing feedback
- Bug reports

## Rules

- Do not commit passwords, API keys, server configs or database dumps.
- Do not push directly to main.
- Keep changes small and focused.
- Explain what your change does in the Pull Request.
